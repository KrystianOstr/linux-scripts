# Bash Mini Projects

## Description

A collection of simple Bash scripts created to practice Linux, scripting, and basic DevOps tasks.

Each project focuses on a small real-world use case such as working with logs, backups, disks, services, or simple deployment automation.

---

## Purpose

This repository was created to improve practical Bash skills through small CLI-based projects.

The main focus is on:

* automation
* working with Linux commands
* scripting logic
* problem solving
* building simple tools for administration tasks

---

## Technologies

* Bash
* Linux CLI
* systemctl
* tar
* grep
* awk
* du
* df
* lsblk
* curl

---

## How to use

Go to the selected project directory, make the script executable, and run it.

```bash
chmod +x script.sh
./script.sh
```
